# fo-dicom-samples

*A Simple DICOM Toolkit*

## 功能

- Simple Dicom Dump [**慎用修改和保存功能**]

- Worklist SCU with MPPS [测试功能]

- Query Retrieve SCU [测试功能]

- C-Store SCU/SCP

- Print SCU/SCP

## 命令行参数

客户端模式 ["-c", "/c", "--client"]

服务端模式 ["-s", "/s", "--server"]
